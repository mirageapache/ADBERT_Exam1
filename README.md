# ADBERT 面試題

## 環境設置

- npm - 8.5.0
- react - 18.2.0
- @mui/material - 5.13.6
- @emotion/react - 11.11.1

## 開始使用

1. Clone Repository

```
git clone https://github.com/mirageapache/ADBERT_Exam1.git
```

2.安裝 npm

```
npm install
```

3.啟動程式

```
npm start
```

4.瀏覽網頁

```
瀏覽器將自動開啟網頁，如未正常開啟請輸入http://localhost:3000/，即可使用
```

5.終止程式

```
在終端機按下兩次 ctrl + C 即可終止程式
```
